## Features

- Multi-threaded token checking
- Configurable thread count and timeout
- Customizable output format


## Installation

1. Download source code

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python main.py
```

1. Load a file with tokens through the interface
2. Configure checking parameters (optional)
3. Start the checking process
4. Results are saved to `results.txt`
